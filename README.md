# Analysis of Factors Affecting Match Outcomes in League of Legends

Statistical analysis of factors affecting match outcomes in League of Legends Season 15 ranked games.

## Data

Dataset: [League of Legends Ranked Match Data (2025)](https://www.kaggle.com/datasets/jakubkrasuski/league-of-legends-ranked-match-data-season-15)

## How to run

```bash
pip install -r requirements.txt
python src/run_all.py
```

## Report

The full analysis and results are described in `report/report.pdf`.
